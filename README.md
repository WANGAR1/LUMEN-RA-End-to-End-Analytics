## Project Overview

Lumera is a Digital Allyship Toolkit designed to equip men with 
practical, trauma-informed skills to support women experiencing 
gender-based violence. This repository contains the complete 
end-to-end data analytics work built to measure the platform's 
learning effectiveness and user impact.

---

## Repository Structure

- **phase_1_data_warehouse** — Schema design, synthetic data 
  generation, and SQL Server warehouse setup
- **phase_2_adhoc_analysis** — SQL queries answering one-off 
  business questions from the warehouse
- **phase_3_dashboards** — Interactive analytics dashboards 
  (Power BI and HTML versions)
- **phase_4_machine_learning** — Predictive models for user 
  behaviour and completion outcomes
- **documentation** — Full project documentation in Word and 
  Markdown formats
