# Phase 3 — Dashboards

## Overview
Two interactive analytics dashboards built to visualize platform 
performance and user retention insights.

## Dashboard 1 — Platform Analytics
Answers questions about learning effectiveness and engagement.

**KPIs:** 39 learners · 111 activities · 41 completions · 
62.9 avg score · 51.8% avg progress

## Dashboard 2 — Retention and Impact
Answers questions about user return rates and drop-off behaviour.

**KPIs:** 79% return rate · 3.2 avg sessions · 18 drop-offs · 
99.2 top score
